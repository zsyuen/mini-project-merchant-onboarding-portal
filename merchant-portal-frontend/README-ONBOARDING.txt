Merchant Onboarding Portal (Angular 20 + Spring Boot API)
=======================================================

Commands:
----------
npm install
npm start

Pages:
-------
- /merchant/register       : Merchant application form
- /merchant/status         : Check application status by reference ID
- /officer/dashboard       : Officer dashboard (view/filter applications)
- /officer/register-admin  : Register new admins
- /officer/manage-admins   : Manage admin privileges

API Base:
----------
http://localhost:8080/api

Spring Boot must be running for backend services (applications, admins, face verification).
